# E-Commerce Arbitrage ETL Pipeline

Скрипт для автоматического расчета юнит-экономики при арбитраже товаров между 1688.com и Kaspi.kz. 
Пайплайн собирает данные, стандартизирует наименования товаров и рассчитывает чистую прибыль с учетом логистики, платформенных сборов и банковских комиссий конвертации.

## Стек технологий
* **Python 3.10+**: Pandas, NumPy, requests (через `curl_cffi`).
* **LLM API**: Google Gemini (gemini-2.5-flash).
* **База данных**: PostgreSQL.
* **BI**: Tableau.

## Дашборд
[Интерактивный дашборд в Tableau Public](https://public.tableau.com/views/E-Commerce_Arbitrage_Unit_Economics/Dashboard1?:language=en-US&publish=yes&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link)

Дашборд включает:
1. **Абсолютная прибыль**: Гистограмма чистой маржи по SKU.
2. **Структура затрат**: Декомпозиция розничной цены (закупка, логистика, комиссии маркетплейса).
3. **Матрица ROI**: Точечная диаграмма (Затраты vs Прибыль) для выявления оптимальных предложений от китайских поставщиков.

## Архитектура и логика работы

### 1. Extract
* Сбор актуальных розничных цен с маркетплейса Kaspi. Для обхода Cloudflare WAF применяется библиотека `curl_cffi`, эмулирующая TLS-отпечатки браузера Chrome.
* Запрос текущего биржевого курса CNY/KZT через открытый API.

### 2. Transform (Entity Resolution)
* Нормализация неструктурированных названий китайских поставщиков в стандартизированные SKU с помощью Gemini API. 
* Для обхода лимитов API и ускорения конвейера реализовано локальное кэширование ответов в JSON.

### 3. Transform (Unit Economics)
Расчет полной себестоимости и маржинальности базируется на следующей модели:
* **Базовая цена (CNY)**: Увеличивается на 13.8% (НДС 1688) и 3% (эквайринг Alipay).
* **Конвертация валют**: Биржевой курс CNY/KZT корректируется на 6.8% для учета банковского спреда при оплате с локальных карт.
* **Международная логистика (Карго)**: 1500 KZT за 1 кг (применяется защитный минимальный порог расчетного веса в 0.4 кг).
* **Локальная доставка**: Фиксированная ставка 1300 KZT.
* **Комиссия маркетплейса (Kaspi) на электронику**: 10.9% от розничной цены.
* *Примечание*: Фиксированный налог ИП (OPEX) исключен из прямой себестоимости единицы товара для корректного расчета чистой маржинальности (COGS).

### 4. Load
* Сохранение обработанных данных в реляционную базу PostgreSQL через SQLAlchemy.
* Выгрузка CSV-файла для обновления дашборда.

## Инструкция по запуску

1. Клонируйте репозиторий:
   ```bash
   git clone [https://github.com/yereke04/arbitrage_etl.git](https://github.com/yereke04/arbitrage_etl.git)
   cd arbitrage_etl

2. Установите зависимости:
   Bash
   pip install -r requirements.txt

3. Создайте конфигурационный файл .env на основе .env.example:
   Ini, TOML
   DB_URL=postgresql://username:password@localhost:5432/arbitrage_db
   GEMINI_API_KEY=your_gemini_api_key
   
4. Запустите конвейер:
   Bash
   python main.py
